# 🚀 FastAPI E-Commerce Store - Day 1 Assignment

## ✨ Features

- ✅ **Product Catalog Management** - View all products with details
- ✅ **Category Filtering** - Filter products by category (Electronics, Stationery)
- ✅ **Stock Management** - View only available/in-stock products
- ✅ **Product Search** - Case-insensitive keyword search
- ✅ **Store Analytics** - Get store summary with statistics
